CSS의 선택자
	- 기본 선택자
	----------------------------------------------------------------------------------------------------------------------------
	선택자		작성예			설명(예시를 기반으로)
	----------------------------------------------------------------------------------------------------------------------------
	*		* {style}		모든 엘리먼트를 선택
	tag		p {style}		태그명이 p인 모든 엘리먼트를 선택
	.class		.hot {style}		class 속성값에 hot을 포함하는 모든 엘리먼트를 선택(class=".., hot, ..")
	#id		#wrapper {style}	id 속성값이 wrapper인 엘리먼트를 선택(id="wrapper")
	tag, tag	div, p {style}		모든 div, 모든 p 엘리먼트를 선택
	.class, .class	.hot, .new {style}	class 속성값에 hot 또는 new를 포함하는 모든 엘리먼트를 선택
	tag.class	div.hot {style}		태그명이 div인 엘리먼트들 중에서 class 속성값에 hot을 포함하는 모든 엘리먼트를 선택
	.class.class	.hot.new {style}	class 속성값에 hot과 new를 함께 포함하고 있는 모든 엘리먼트를 선택
	----------------------------------------------------------------------------------------------------------------------------
	* 기본 선택자는 모두 암기할 것
	

	- 조합 선택자 (공백, >, +, ~ 등의 기호를 이용해 2개 이상의 기본 선택자를 조합한 것)
	----------------------------------------------------------------------------------------------------------------------------
	선택자		작성예			설명(예시를 기반으로)
	----------------------------------------------------------------------------------------------------------------------------
	공백(후손 선택자)
	tag tag		div p			<div>의 후손 중에서 모든 <p>를 선택
	.class .class	.news .hot		class에 news를 포함하고 있는 모든 엘리먼트의 후손 중에서 class에 hot을 포함하고 있는 모든 엘리먼트를 선택
	.class tag	.news p			class에 news를 포함하고 있는 모든 엘리먼트의 후손 중에서 모든 <p>를 선택
	#id tag		#footer p 		id가 footer인 엘리먼트의 후손 중에서 모든 <p> 선택
	#id .class	#header .menu		id가 header인 엘리먼트의 후손 중에서 class에 menu를 포함하고 있는 모든 엘리먼트를 선택

	> (자식 선택자) * 이하 위 5가지의 조합 중 하나만 예시로 작성한다.
	tag > tag	div > p			<div>의 자식 중에서 모든 <p>를 선택

	+ (인접한 형제 선택자)
	tag + tag	h3 + p			h3 바로 다음에 있는 p를 선택

	~ (형제 선택자)
	tag ~ tag	h3 ~ p			h3 다음에 있는 모든 p를 선택
	----------------------------------------------------------------------------------------------------------------------------

	- 속성 선택자
	----------------------------------------------------------------------------------------------------------------------------
	선택자		작성예			설명(예시를 기반으로)
	----------------------------------------------------------------------------------------------------------------------------
	[attr]		[name]			name 속성을 가진 모든 엘리먼트를 선택
	[attr="value"]	input[type="text"]	input 태그 중에서 type 속성이 text인 모든 엘리먼트를 선택
	[attr^="val"]	a[href^="http"]		a태그 중에서 href 속성이 http로 시작하는 모든 엘리먼트를 선택
	[attr$="val"]	a[href$="net"]		a태그 중에서 href 속성이 net으로 끝나는 모든 엘리먼트를 선택
	[attr*="val"]	a[title*="학교"]	a태그 중에서 title 속성값에 "학교"라는 문자를 포함하고 있는 모든 엘리먼트 선택	
	[attr~="val"]	a[title~="학교"]	a태그 중에서 title 속성값에 "학교"라는 문자를 공백으로 구분된 단어로 포함하고 있는 모든 엘리먼트 선택
	----------------------------------------------------------------------------------------------------------------------------


	- 필터 선택자
	----------------------------------------------------------------------------------------------------------------------------
	선택자		작성예			설명(예시를 기반으로)
	----------------------------------------------------------------------------------------------------------------------------
	:first-child	p:first-child		p 중에서 부모를 기준으로 첫번째 자식 엘리먼트인 p를 선택
	:last-child	p:last-child		p 중에서 부모를 기준으로 마지막 자식 엘리먼트인 p를 선택
	:nth-child(n)	p:nth-child(2)		p 중에서 부모를 기준으로 두 번째 자식 엘리먼트인 p를 모두 선택
			p:nth-child(odd)	p 중에서 부모를 기준으로 홀수번째 자식 엘리먼트인 p를 모두 선택
			p:nth-child(even)	p 중에서 부모를 기준으로 짝수번째 자식 엘리먼트인 p를 모두 선택
	:not(selector)	p:not(:first-child)	p 중에서 부모를 기준으로 첫번째 자식이 아닌 p를 모두 선택
	----------------------------------------------------------------------------------------------------------------------------
	* :는 띄어쓰기를 하지 않고 붙여쓴다
	* div :first-child : div 안에서 부모를 기준으로 첫번째 자식인 엘리먼트를 선택
	* div:first-child : div중에서 부모를 기준으로 첫번째 자식 엘리먼트인 div를 선택


	- 가상 선택자
	----------------------------------------------------------------------------------------------------------------------------
	선택자		작성예			설명(예시를 기반으로)
	----------------------------------------------------------------------------------------------------------------------------
	:hover		a:hover			a 중에서 마우스포인터가 올라가 있는 것을 선택
	:link		a:link			a 중에서 방문한 적이 없는 링크
	:visited	a:visited		a 중에서 방문한 적이 있는 링크
	:active		a:active		a 중에서 지금 클릭하는 링크

	:hover		div:hover		div중에서 마우스포인터가 올라가 있는 것 선택
