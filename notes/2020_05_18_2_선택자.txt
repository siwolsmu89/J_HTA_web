CSS의 선택자
	- 기본 선택자
	----------------------------------------------------------------------------------------------------------------------------
	선택자		작성예			설명(예시를 기반으로)
	----------------------------------------------------------------------------------------------------------------------------
	*		* {style}		모든 엘리먼트를 선택
	tag		p {style}		태그명이 p인 모든 엘리먼트를 선택
	.class		.hot {style}		class 속성값에 hot을 포함하는 모든 엘리먼트를 선택(class=".., hot, ..")
	#id		#wrapper {style}	id 속성값이 wrapper인 엘리먼트를 선택(id="wrapper")
	tag, tag	div, p {style}		모든 div, 모든 p 엘리먼트를 선택
	.class, .class	.hot, .new {style}	class 속성값에 hot 또는 new를 포함하는 모든 엘리먼트를 선택
	tag.class	div.hot {style}		태그명이 div인 엘리먼트들 중에서 class 속성값에 hot을 포함하는 모든 엘리먼트를 선택
	.class.class	.hot.new {style}	class 속성값에 hot과 new를 함께 포함하고 있는 모든 엘리먼트를 선택
	----------------------------------------------------------------------------------------------------------------------------
	* 기본 선택자는 모두 암기할 것
	
	- 조합 선택자 (공백, >, +, ~ 등의 기호를 이용해 2개 이상의 기본 선택자를 조합한 것)
	----------------------------------------------------------------------------------------------------------------------------
	선택자		작성예			설명(예시를 기반으로)
	----------------------------------------------------------------------------------------------------------------------------
	공백(후손 선택자)
	tag tag		div p			<div>의 후손 중에서 모든 <p>를 선택
	.class .class	.news .hot		class에 news를 포함하고 있는 모든 엘리먼트의 후손 중에서 class에 hot을 포함하고 있는 모든 엘리먼트를 선택
	.class tag	.news p			class에 news를 포함하고 있는 모든 엘리먼트의 후손 중에서 모든 <p>를 선택
	#id tag		#footer p 		id가 footer인 엘리먼트의 후손 중에서 모든 <p> 선택
	#id .class	#header .menu		id가 header인 엘리먼트의 후손 중에서 class에 menu를 포함하고 있는 모든 엘리먼트를 선택

	> (자식 선택자) * 이하 위 5가지의 조합 중 하나만 예시로 작성한다.
	tag > tag	div > p			<div>의 자식 중에서 모든 <p>를 선택

	+ (인접한 형제 선택자)
	tag + tag	h3 + p			h3 바로 다음에 있는 p를 선택

	~ (형제 선택자)
	tag ~ tag	h3 ~ p			h3 다음에 있는 모든 p를 선택
	----------------------------------------------------------------------------------------------------------------------------
