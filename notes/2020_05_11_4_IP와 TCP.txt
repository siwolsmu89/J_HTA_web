IP(Internet Protocol)
	- ip 주소체계를 규정하는 프로토콜
	- 종류
		- IPv4
			* 32비트로 구성된다.
			* 총 2^32개(4,294,967,296개)의 주소를 가진다.
			* 클래스
				클래스 A 주소 : 1.xxx.xxx.xxx ~ 126.xxx.xxx.xxx		256^3개의 host를 구성가능
				클래스 B 주소 : 128.???.xxx.xxx ~ 191.???.xxx.xxx	256^2개의 host를 구성가능
				클래스 C 주소 : 192.???.???.xxx ~ 223.???.???.xxx	256^1개의 host를 구성가능
				* 클래스는 연결가능한 host의 개수에 따라서 구분된다.
			* 구분
				공인 ip주소 : 다른 컴퓨터와 통신할 때 사용하는 ip주소
				사설 ip주소 : 가상 ip, 사설망(내부망)에서 사용할 수 있는 ip주소
					      망 외부로 나가는 것은 가능하지만, 외부에서 망 내로는 접근이 불가능함
					클래스 A 주소 : 10.0.0.0 ~ 10.255.255.255
					클래스 B 주소 : 172.16.0.0 ~ 172.31.255.255
					클래스 C 주소 : 192.168.0.0 ~ 192.168.255.255
				특수 목적 ip주소
					127.0.0.1 : 루프백(loopback)ip, host 자신을 나타내는 ip주소
		- IPv6	

TCP(Transmission Control Protocol)
